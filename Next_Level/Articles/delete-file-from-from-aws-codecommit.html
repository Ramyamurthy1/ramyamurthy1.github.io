How to delete a file from AWS Code Commit

Deleting a file is not easy or straight forward job. It can cause a permanent data loss situation. AWS provides extra checks to make sure you know what you are doing. The below code explains how to delete a file from AWS Code Commit repository. You can use AWS CLI to perform these steps. First, get the Code Commit repository Commit id. This will give us the repository commit details. This can help identify the changes made, which include,

  • When and who made the change
  • When the commits were merged to the branch

To get the commit id run the below command, 

                
                   aws codecommit get-branch --repository-name SampleRepo --branch-name test-branch

The out Sample Output:

                
                    
                    {
                        "branch": {
                                    "branchName": "main",
                                    "commitId": "xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
                                  }
                    }

The comminId describes the full SHA ID of the specified commit. The commit id is then provided to actually delete file. 

                
                    aws codecommit delete-file --repository-name SampleRepo --branch-name test-branch --file-path file-path --parent-commit-id xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
